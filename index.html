Notes: Algernon Blackwood, Short Fiction

Suggested ways to improve this collection are listed as follows, listed in approximate order of expected return for investment of effort.

  1. Read the included works from source [5], indicated as not read, i.e. not in bold.
  2. Read the remaining included works, indicated as not read.
  3. Transcribe untranscribed words for which page scans are available.
  4. Search for public domain page scans of the transcribed works for which only non-PD page are shown.
  5. Search for works published in 1925.
  6. Read the works marked as read to find errors.
  7. Locate sources for the works for which no source is know, e.g. in a physical library.
